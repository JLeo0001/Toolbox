# 安卓工具箱 (Android Tools)

一个功能强大的安卓工具类应用，包含网络端口扫描和文件合并拆分功能。

## 功能特性

### 🌐 TCP端口扫描器
- 支持输入任意网址或IP地址
- 可配置端口范围扫描
- 支持自定义超时时间和并发线程数
- 提供扫描历史记录功能
- 实时显示扫描进度和结果

### 📁 文件合并拆分器
- 兼容MERGEDv3格式的文件合并与拆分
- 支持视频文件与附件的隐藏合并
- 提供内存模式和流式模式处理
- 包含开发者调试模式
- 美观的Material Design 3界面

## 技术栈

- **开发语言**: Kotlin
- **UI框架**: Jetpack Compose
- **架构模式**: MVVM
- **异步处理**: Kotlin Coroutines
- **导航**: Navigation Compose
- **主题**: Material Design 3

## 权限要求

- `INTERNET` - 网络访问权限（端口扫描功能）
- `ACCESS_NETWORK_STATE` - 网络状态访问权限
- `READ_EXTERNAL_STORAGE` - 读取外部存储权限（文件操作）
- `WRITE_EXTERNAL_STORAGE` - 写入外部存储权限（文件操作）

## 安装要求

- Android 7.0 (API level 24) 或更高版本
- 至少 50MB 可用存储空间

## 使用说明

### 端口扫描
1. 在首页点击"TCP端口扫描器"
2. 输入目标主机名或IP地址
3. 设置端口范围（起始端口-结束端口）
4. 配置超时时间和并发线程数
5. 点击"开始扫描"按钮
6. 查看实时扫描结果和历史记录

### 文件合并
1. 在首页点击"文件合并拆分器"
2. 切换到"合并"标签页
3. 选择视频文件和要隐藏的附件文件
4. 选择处理模式（内存模式/流式模式）
5. 点击"开始合并"
6. 等待处理完成并查看结果

### 文件拆分
1. 在文件合并拆分器中切换到"拆分"标签页
2. 选择要拆分的MERGEDv3格式文件
3. 点击"开始拆分"
4. 等待处理完成并获取原始文件

## 开发者模式

在文件合并拆分器中，可以启用开发者模式来查看详细的调试信息：
- 文件处理过程日志
- 错误详情和堆栈跟踪
- 性能统计信息

## 注意事项

1. 端口扫描功能请遵守相关法律法规，仅用于合法的网络诊断
2. 大文件处理时建议使用流式模式以避免内存不足
3. 确保设备有足够的存储空间进行文件操作
4. 某些防火墙可能会阻止端口扫描活动
